
# PyTorch Fundamentals

A collection of notebooks covering basic PyTorch concepts.

## Contents

### `Basics.ipynb`
- Tensor creation and operations
- Matrix multiplication and transposition
- Broadcasting with tensors
- Batch matrix multiplication
- Reshaping tensors

### `grad.ipynb`
- Autograd with scalar values
- Partial derivatives computation
- Chain rule implementation
- Automatic differentiation with vectors

## Requirements

```bash
pip install torch
```

## Usage

Open notebooks with Jupyter:

```bash
jupyter notebook
```
